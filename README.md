# 抽象 · chouxiang

> 一个用来收藏「抽象文化」的开源资料库：起源、黑话、名场面、语言构造法，外加一个能现场整活的 **[抽象话生成器](tools/抽象话生成器.html)**。

「抽象」原本是哲学 / 艺术里「抽取本质」的词。但在中文互联网，它有了另一层意思——

2015 年，主播 **李赣** 在直播间里一口一个「真是太抽象了」，从此把「抽象」和一种 **违背常理、解构主流、无厘头、用荒诞消解现实** 的亚文化绑在了一起。十年过去，「抽象」从斗鱼 6324 直播间的嘴臭互喷，变成了小红书 **2024 年的年度关键词**。

本项目试图把这套「抽象文化」的来龙去脉、黑话梗、名场面、语言构造法，整理成一个 **开源、可贡献、可持续更新** 的资料库。欢迎一起来整活。
