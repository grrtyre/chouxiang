# 抽象 · chouxiang

> 一个**纯粹的知识搜集**仓库：记录中文互联网「抽象文化」里的那些东西和梗，每一条都尽量讲清楚它**怎么来的**。

---

## 这是什么？

「抽象」原本是哲学 / 艺术里「抽取本质」的词。但在中文互联网，它有了另一层意思——2015 年主播李赣在直播间一口一个「真是太抽象了」，把「抽象」和一种 **违背常理、解构主流、无厘头、用荒诞消解现实** 的亚文化绑在了一起。十年过去，它从斗鱼 6324 直播间的嘴臭互喷，变成了小红书 2024 年的年度关键词。

本项目 **不整花活**，只做一件事：把这套文化里的梗、人物、名场面、黑话，一条条搜集、整理、标注来源，做成一个 **可考、可贡献** 的知识库。
