# 抽象黑话辞典

> 把长评价压成单字、把脏话换成 emoji、把大白话改成谐音——这就是抽象话的精髓。下面按类别整理，尽量标出处。

---

## 一、核心单字（抽象话的「压缩包」）

| 黑话 | 含义 | 出处 / 用法 |
| --- | --- | --- |
| **典** | 「典型」的浓缩，指某人言行是某类人的典型代表（多带戏谑 / 贬义） | 「典中典」= 典型中的典型 |
| **绷** | 源自「绷不住了」，指忍俊不禁、笑到破防 | 「绷带」「蚌埠住了」同源 |
| **孝** | 「孝子」的异化，指对某人或事物过度拥护、盲目维护（常反讽） | 「老孝子」「大孝子」 |
| **急** | 形容对方被激怒、焦虑、破防的状态 | 「急了」「急眼」 |
| **破防** | 心理防线被击破，情绪失控 | 通用网络梗，抽象圈高频 |
| **整活** | 搞事情、制造节目效果、玩出花样 | 「一切皆可解构，万物皆可抽象」 |
| **乐子** | 娱乐、看热闹 | 「乐子人」「找乐子」 |

## 二、人物与名号（抽象圈的「反偶像」）

| 名号 | 是谁 | 标签 |
| --- | --- | --- |
| **李老八 / 李赣** | 抽象工作室创始人 | 「真是太抽象了」命名者 |
| **孙笑川** | 抽象工作室核心 | 早期「嘴臭」代表；「孙吧（孙笑川吧）」是抽象文化重要阵地 |
| **带篮子（陈义）** | 主播 | 「早安打工人」「大专人」输出者 |
| **药水哥（刘波）** | 主播 | 名场面「您配吗」对喷 7 小时 |
| **战鹰** | 围棋棋手 | 后来被归入抽象「反偶像」之一 |
| **王境泽** | 综艺人物 | 「真香定律」本尊 |

## 三、群体自称（自嘲是抽象的魂）

| 词 | 含义 |
| --- | --- |
| **鼠鼠** | 「下水道的鼠鼠」，自嘲底层、卑微的年轻人 |
| **牛马** | 「大厂牛马」，自嘲打工人卑微 |
| **大专人 / 带专人** | 带篮子相关，自嘲学历（实际学历未必是大专） |
| **打工人** | 统一白领与体力劳动者的自嘲称呼，2020 年度流行语 |

## 四、谐音梗（读音相近，字换掉）

| 抽象说法 | 原意 | 说明 |
| --- | --- | --- |
| **蚌埠住了** | 绷不住了 | 笑到崩溃 |
| **针不戳** | 真不戳（真不错） | 出自某主播口音 |
| **夺笋** | 多损 | 调侃「多损啊」 |
| **集美** | 姐妹 | 出自某主播口音 |
| **尊嘟假嘟** | 真的假的 | 卖萌版疑问 |
| **我嘞个豆** | —— | 表示惊讶的口癖 |
| **芜湖** | 呜呼 | 起飞的欢呼（出自「大司马」） |
| **玉玉** | 抑郁 | 调侃写法 |
| **寄** | 寄了 = 完了 / GG | 谐「寄」 |

## 六、通用网络梗（常与抽象文化混用）

| 词 | 含义 |
| --- | --- |
| **真香** | 起初拒绝、后来打脸（王境泽名场面） |
| **退退退** | 某阿姨吵架动作成梗，表示驱赶 / 拒绝 |
| **雪糕刺客** | 价格离谱的雪糕，隐喻「隐藏高价」 |
| **冤种** | 倒霉 / 冤大头的人 |
| **emo** | 情绪低落 |
| **贴脸开大** | 当面对线、直接攻击 |
| **高产似那啥** | 形容产出极快（那啥 = 母猪，含蓄说法） |
| **疯狂星期四** | 肯德基「V 我 50」梗，群体复制玩梗 |
