# 抽象梗知识库（黑话辞典）

> 本仓库是一个**纯粹的知识搜集**：每一条「抽象的东西」和「梗」都尽量讲清楚 **它怎么来的**。
>
> **抽象话是怎么形成的？（一句话机制）**
> 抽象话主要有五种造词法：① **谐音替换**（绷不住→蚌埠住、真不错→针不戳）；② **emoji / 符号替换**（用表情突破审查）；③ **拼音缩写**（取拼音首字母，如 yyds、xswl）；④ **单字浓缩**（把长评价压成单字：典型→典、绷不住→绷、孝子→孝）；⑤ **旧词新解 / 反讽**（孝子变贬义、真香变打脸）。下面每条都标「怎么来的」。

---

## 一、核心单字（抽象话的「压缩包」）

**典**
- 含义：典型的浓缩，戏谑地指某人言行是某类人的典型代表。
- 怎么来的：源自贴吧 / 直播间的简写习惯，「典型」被压成单字「典」；「典中典」= 典型中的典型。
- 用法：看到某人精准踩中某类人特征时刷「太典了」。

**绷**
- 含义：忍俊不禁、笑到破防。
- 怎么来的：来自「绷不住了」——情绪那根弦「绷」断了；后来又被谐音成「蚌埠住了」。
- 用法：看到离谱内容时「绷不住了」「绷带」。

**孝**
- 含义：对某人 / 事物过度拥护、盲目维护，常带反讽。
- 怎么来的：「孝子」本是褒义，在抽象语境被异化，用来讽刺无脑维护（「老孝子」「大孝子」）。
- 用法：反讽粉圈式护短、无脑吹。

**急**
- 含义：形容对方被激怒、焦虑、破防。
- 怎么来的：直播间对线时形容对方「急了」「急眼」，即被戳中痛处。
- 用法：对方破防时刷「急了」。

**破防**
- 含义：心理防线被击破，情绪失控。
- 怎么来的：本是游戏术语（护甲 / 防御被破），后来泛化为情绪上的崩溃。
- 用法：被戳中痛点，或被感动到崩溃，都叫破防。

**整活**
- 含义：搞事情、制造节目效果、玩出花样。
- 怎么来的：东北方言「整」+「活」，直播间用来形容主播制造效果。
- 用法：「你又在整活」。

**乐子**
- 含义：娱乐、看热闹。
- 怎么来的：抽象文化核心词；「找乐子」「乐子人」指以旁观热闹为乐，不带立场。
- 用法：「纯属图个乐子」。

## 二、人物与名号（抽象圈的「反偶像」）

**李老八 / 李赣**
- 身份：抽象工作室创始人。
- 怎么来的：2015 年 9 月 1 日，在成都街头烧烤摊宣布「抽象工作室」成立；一口一个「真是太抽象了」，把「抽象」和这套亚文化焊死。

**孙笑川**
- 身份：抽象工作室核心。
- 怎么来的：早期「嘴臭」代表；「孙笑川吧（孙吧）」是抽象文化的重要阵地。

**带篮子（陈义）**
- 身份：主播。
- 怎么来的：「早安打工人」「大专人」输出者；2020 年《咬文嚼字》把「打工人」评为年度十大流行语。

**药水哥（刘波）**
- 身份：主播。
- 怎么来的：名场面「您配吗？」——和粉丝对喷 7 小时；常在直播间学狗叫、摇耳朵、吐舌头、歪头做鬼脸，用扮演小丑提供虚无的快乐。李赣评：「非常有才华，且非常不要脸。」

**战鹰**
- 身份：围棋棋手。
- 怎么来的：后来被归入抽象「反偶像」之一，体现抽象圈「反偶像」不断被重写的特点。

**王境泽**
- 身份：综艺人物。
- 怎么来的：「真香定律」本尊——起初信誓旦旦拒绝，最后真香打脸。

## 三、群体自称（自嘲是抽象的魂）

**鼠鼠**
- 含义：自嘲底层、卑微的年轻人。
- 怎么来的：源自「下水道的鼠鼠」，由贴吧自嘲梗演化而来。

**牛马**
- 含义：自嘲打工人像牛马一样被使唤。
- 怎么来的：比喻性自称，常见于「大厂牛马」。

**大专人 / 带专人**
- 含义：自嘲学历（实际学历未必是大专）。
- 怎么来的：带篮子相关；源于他大专毕业时安慰迷茫同学的自拍短视频。

**打工人**
- 含义：统一白领与体力劳动者的自嘲称呼。
- 怎么来的：2020 年带篮子在深圳拍视频感叹「早安，打工人！」；后被《咬文嚼字》评为年度流行语，并泛化成几乎人人可自称。

## 四、谐音梗（读音相近，字换掉）

**蚌埠住了**
- 含义：绷不住了，笑到崩溃。
- 怎么来的：借用安徽地名「蚌埠」谐音「绷不」。

**针不戳**
- 含义：真不戳 = 真不错。
- 怎么来的：某主播口音把「真不错」说成了「针不戳」。

**夺笋**
- 含义：多损啊（调侃）。
- 怎么来的：谐音「多损」。

**集美**
- 含义：姐妹。
- 怎么来的：某主播口音把「姐妹」说成「集美」。

**尊嘟假嘟**
- 含义：真的假的（卖萌疑问）。
- 怎么来的：把「真的假的」萌化发音。

**我嘞个豆**
- 含义：表示惊讶的口癖。
- 怎么来的：网络口癖，无特定单一出处。

**芜湖**
- 含义：呜呼，表示起飞 / 欢呼。
- 怎么来的：出自主播「大司马」的口头禅「呜呼」，谐音成「芜湖」。

**玉玉**
- 含义：抑郁（调侃写法）。
- 怎么来的：谐音「抑郁」。

**寄**
- 含义：寄了 = 完了 / GG。
- 怎么来的：谐音「寄」，表示事情黄了。

## 六、通用网络梗（常与抽象文化混用，每条都标来源）

**真香**
- 含义：起初拒绝、后来打脸。
- 怎么来的：综艺《变形计》里王境泽的名场面。

**退退退**
- 含义：驱赶 / 拒绝的动作梗。
- 怎么来的：网络视频里一位阿姨双手推人连喊「退退退」，被做成表情包。

**雪糕刺客**
- 含义：价格离谱的雪糕，隐喻「隐藏高价」。
- 怎么来的：2022 年夏天「天价雪糕」话题，用「刺客」比喻结账时才发现被狠宰。

**冤种**
- 含义：倒霉 / 冤大头的人。
- 怎么来的：东北方言。

**emo**
- 含义：情绪低落、丧。
- 怎么来的：英文 emotional 的缩略，泛指情绪低落。

**贴脸开大**
- 含义：当面对线、直接攻击。
- 怎么来的：游戏术语（贴脸放大招），引申为当面怼人。

**高产似那啥**
- 含义：形容产出极快。
- 怎么来的：「那啥」是「母猪」的含蓄说法，贴吧梗。

**疯狂星期四**
- 含义：肯德基「V 我 50」梗，群体复制玩梗。
- 怎么来的：肯德基每周四有优惠，网友在周四编段子求请客，形成固定玩法。
