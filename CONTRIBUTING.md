# 贡献指南

这是一个**纯粹的知识搜集**项目：记录抽象文化里的梗、人物、名场面、黑话，并且 **每一条都要讲清楚「怎么来的」**。

## 我能贡献什么？

- **补充梗 / 抽象的东西**：在 [`docs/抽象黑话辞典.md`](docs/抽象黑话辞典.md) 里按现有格式加一条，务必带「怎么来的」。
- **补全来源**：任何条目都可以补充更准确的 **出处 / 来源人物 / 时间**。
- **丰富名场面**：在 [`docs/经典语录与名场面.md`](docs/经典语录与名场面.md) 里加经典台词或事件，并说明它怎么来的。
- **纠错**：考据错了就提 PR 改。
- **加入感谢名单**：在 [`SPONSORS.md`](SPONSORS.md) 里加一行（如果你在爱发电支持了本项目）。

## 提交方式

1. Fork 本仓库
2. 新建分支：`git checkout -b feat/你的贡献`
3. 提交改动：`git commit -m "docs: 补充 xx 梗"`
4. 推送并发起 Pull Request

也欢迎直接开 Issue 提建议或纠错。

## 内容规范

- **每条都要标「怎么来的」**：出处、来源人物、时间，方便后人考据。
- 释义用大白话，别整得比抽象还抽象。
- 尊重事实，避免对具体人物的过度攻击；这是文化资料库，不是对线现场。

感谢每一个来整活的人 🤡
