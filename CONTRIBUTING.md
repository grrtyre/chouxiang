# 贡献指南

欢迎一起来整活！这个项目靠大家共同维护才能越来越「抽象」。

## 我能贡献什么？

- **补充黑话 / 梗**：在 [`docs/抽象黑话辞典.md`](docs/抽象黑话辞典.md) 里按现有格式加一条。
- **纠错 / 补出处**：任何条目都可以补充更准确的来源、用法、时间线。
- **丰富名场面**：在 [`docs/经典语录与名场面.md`](docs/经典语录与名场面.md) 里加经典台词或事件。
- **改进生成器**：直接改 [`tools/抽象话生成器.html`](tools/抽象话生成器.html)，加更多谐音 / emoji 替换规则。
- **加入感谢名单**：在 [`SPONSORS.md`](SPONSORS.md) 里加一行（如果你在爱发电支持了本项目）。

## 提交方式

1. Fork 本仓库
2. 新建分支：`git checkout -b feat/你的贡献`
3. 提交改动：`git commit -m "docs: 补充 xx 梗"`
4. 推送并发起 Pull Request

也欢迎直接开 Issue 提建议或纠错。

## 内容规范

- 尽量标注梗的 **出处 / 来源人物 / 时间**，方便后人考据。
- 释义用大白话，别整得比抽象还抽象（不是）。
- 尊重事实，避免对具体人物的过度攻击；这是文化资料库，不是对线现场。
- 生成器里的替换词保持「温文儒雅」，别往脏话方向整。

感谢每一个来整活的人 🤡
